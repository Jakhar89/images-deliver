⚡ How jsDelivr Works

jsDelivr proxies files directly from GitHub repositories and serves them from 750+ global PoP nodes with automatic caching and HTTP/2 support.

📤 How to Add New Images

  1. Add your image file to the images/ folder in this repository.
  2. Run git add images/your-image.png
  3. Commit: git commit -m "add: new image"
  4. Push: git push origin main
  5. Copy the CDN URL from the dashboard above — it's live in seconds!

🌐 Use in HTML

Reference the CDN URL directly inside an <img> tag or as a CSS background-image:

HTML 
<img
  src="https://cdn.jsdelivr.net/gh/user/repo@main/images/hero.png"
  alt="Hero image"
  loading="lazy"
/>

⚛️ Use in React / Next.js

Import or inline the CDN URL as a constant for clean, maintainable code.

JSX 
const CDN = "https://cdn.jsdelivr.net/gh/user/repo@main";

export default function Hero() {
  return (
    <img
      src={`${CDN}/images/hero-cyberpunk.png`}
      alt="Hero"
    />
  );
}
