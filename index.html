Upload Images via GitHub API

No CLI or local setup needed. Images upload directly from your browser to GitHub and are instantly available via CDN.

🔑 GitHub Personal Access Token

Not set

Required once. Your token is saved only in your browser's localStorage — never sent anywhere except directly to the GitHub API.

How to get a token:

  1. Go to github.com/settings/tokens/new
  2. Set a note (e.g. "ImageCDN Hub"), set expiry
  3. Under Scopes, tick repo
  4. Click Generate token — copy it immediately
✅ Token saved securely in localStorage

Host Images. Ship Faster.

Drop your images into this repository, commit and push to GitHub, and instantly serve them globally through jsDelivr's massive CDN network — free, fast, and forever cached.

⚡ How jsDelivr Works

jsDelivr proxies files directly from GitHub repositories and serves them from 750+ global PoP nodes with automatic caching and HTTP/2 support.

https://cdn.jsdelivr.net/gh/:user/:repo@:version/:path https://cdn.jsdelivr.net/gh/jakhar/images-deliver@main/images/shared/hero.png

📤 Upload Without CLI

  1. Go to the Upload tab above.
  2. Save your GitHub PAT (repo scope required).
  3. Choose the site folder, drop your images.
  4. Click Upload All to GitHub.
  5. Copy the CDN URL that appears instantly!
